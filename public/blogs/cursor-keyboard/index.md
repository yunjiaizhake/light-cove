
**Ctrl B**是打开左边文件面板，右侧**AI 面板**设置为 **Ctrl + Shift + B** 比较合适 （原本是 ctrl+alt+b, ctrl+shift+b 原本是打开浏览器）

经常打开**Terminal**，也设置一下，使用 **Ctrl + ~** （有可能是默认，可以检查下）

## 常用快捷键

* `Alt + ↑ / ↓`

整行代码上下移动。

* `Shift + Alt + ↓`

复制当前行到下一行。

## 其它

* 单独想放大左侧目录字体大小

没有单独设置，但是可以先全局放大 `"window.zoomLevel": 0.5`，再单独缩小 editor 文字大小 `"editor.fontSize": 12`
